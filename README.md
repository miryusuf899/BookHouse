# 📘 BookHouse — Online Library

**BookHouse** — это современное веб-приложение на Django для управления библиотечным каталогом. Проект реализован в рамках экзамена по теме **Django MVT & CRUD (Function Based Views)**.

## 🚀 Основные возможности (Features)
* **Полный CRUD** для 4 моделей: Книги, Авторы, Категории и Издательства.
* **Связи в БД**: Использование ForeignKey для организации структуры "книга — автор/категория/издательство".
* **Медиа**: Загрузка и отображение обложек книг, логотипов издательств и фото авторов.
* **UI/UX**: Современный темный интерфейс с анимированным Dashboard и кастомными формами.
* **Static & Media**: Полная настройка статических и медиа-файлов.

## 🛠 Технологии
* **Backend:** Python 3.x, Django 5.x / 6.x
* **Frontend:** HTML5, CSS3 (Custom animations), FontAwesome icons
* **Database:** SQLite3
